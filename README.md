## hosted on render
- API - https://booking-backend-bofk.onrender.com

---

## Features

- Create a booking (customer)
- Automatic provider assignment
- Provider workflow (accept / reject)
- Booking lifecycle  
  pending → assigned → in-progress → completed
- Failure handling  
  cancellations, rejections, no-shows
- Retry logic for reassignment
- Admin / Ops override panel
- Booking history and event logs
- Simple multi-screen frontend with routing

---

## Tech Stack

**Backend**
- Node.js, Express
- MongoDB, Mongoose
